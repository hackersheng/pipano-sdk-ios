# PiPanoSDK-iOS
## 简介

PiPanoSDK 是一套用于处理全景图像的开发工具包，包含以下功能：

1. 支持 浏览图片、播放视频、预览视频流。
2. 支持图像输入源包含：单鱼眼，双鱼眼，全景2:1等；
3. 支持多种展开模式：沉浸、鱼眼、小行星、圆柱、VR、双画卷等；
4. 支持多种滤镜效果：锐化、木炭笔、轮廓、蓝莓、像素化等；
5. 支持多种过场动画：翻转、渐变、开门、光圈、折叠等；
